Leaf Disease Detection Project

1. Install Python 3.11
2. Open terminal in this folder.
3. Create venv:
   python -m venv venv
4. Activate:
   venv\Scripts\activate
5. Install:
   pip install -r requirements.txt
6. Download PlantVillage dataset and place class folders inside dataset/
7. Train:
   python train.py
8. Test:
   python predict.py image.jpg
9. Webcam:
   python webcam.py
10. Web:
   python app.py
